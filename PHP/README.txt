Introduction.
This project shows some examples in PHP language using XAMPP.

Some Prerequisites: 
- Set your enviroment by installing XAMPP.
	If you are in Windows, you can follow the next link: http://www.wikihow.com/Install-XAMPP-for-Windows
	If you are in Linux, you can follow the next link: http://www.wikihow.com/Install-XAMPP-on-Linux
	If you are in OS X, you can follow the next link: http://vichargrave.com/how-to-install-xampp-on-mac-os-x-step-by-step/

Example Projects:
-> simpleDBConnection: This example shows how to create a DB Connection using PHP and show the retrieved information in a HTML Page.
	NOTE: To run this project properly, you should do the next:
		* You should import the School.sql database into your Database Management (presumably the MySQl within the XAMPP).
		   You can follow the next link for that: http://www.zymphonies.com/blog/how-import-and-export-database-using-phpmyadmin
		* You should place the correct db connection variables (username, password, servername) in the connectdb.php file.
